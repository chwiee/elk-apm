# elk-apm

## Requisitos
| Nome | Versão |
| :--: | :----: |
| Docker | 19.03 + |
| Compose| 01.18 + |
| Git    | --      |

## Como usar
Basta fazer o git clone do projeto e executar o docker-compose da raiz do projeto com o comando

```bash
docker-compose up -d
```

Aguarde todos os processos terminarem e use

```bash
docker ps 
```

Para checar todos os containers que estão UP
