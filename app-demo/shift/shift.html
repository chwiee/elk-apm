Docker é a principal tecnologia para execução de contêineres (container runtime) utilizado por mais de 80% das empresas para fornecimento de microserviços em ambiente cloud ou on-premises para serviços WEB, Mobile e IoT (Fonte: Container Orchestrators in Sysdig’s 2018 Docker Usage Report). Necessária para ampliar a colaboração e integração entre os Desenvolvedores e a equipe de Infraestrutura, a tecnologia de contêineres passou a ser utilizada como uma base nos fluxos CI/CD das práticas DevOps.
